## One-line project description
Designed and built a production-style Job Portal backend using Spring Boot with layered architecture, JWT authentication, MySQL persistence, validation, and global exception handling.

## Resume bullets (pick 3-4)
- Developed a RESTful backend in Java/Spring Boot using Controller-Service-Repository architecture to improve maintainability and separation of concerns.
- Implemented JWT-based authentication with BCrypt password hashing for secure user registration and login workflows.
- Built complete Job CRUD APIs with DTO-based request/response models and input validation to enforce API contract correctness.
- Added centralized exception handling (`@RestControllerAdvice`) with standardized error payloads for predictable client integration.
- Integrated Spring Data JPA with MySQL and configured Hibernate auto schema updates for rapid development.
- Documented APIs with Swagger/OpenAPI for faster testing and developer onboarding.

## ATS-friendly tech stack line
Java, Spring Boot, Spring Security, Spring Data JPA, REST APIs, JWT, BCrypt, MySQL, Hibernate, Maven, Swagger/OpenAPI, Exception Handling, DTO Validation

## Interview-ready talking points
- Why service layer: keeps controllers thin and moves business logic to reusable units.
- Why DTOs: avoids leaking entity internals and enforces clear API contracts.
- Why global exception handling: consistent client-facing errors and cleaner controllers.
- Why BCrypt: avoids storing plaintext passwords and follows basic secure auth practices.
